# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

## [0.2.0] - 2026-04-10

### Added
- Dark mode as default theme
- Light/dark mode toggle (sun/moon icon in navbar) via `docsify-darklight-theme`
- Theme preference persisted in `localStorage`

## [0.1.0] - 2026-04-10

### Added
- Initial Docsify site setup with GitHub Pages support
- Landing page (`README.md`) with community disclaimer
- Sidebar navigation for all 23 documented endpoints
- Search plugin
- 23 endpoint specs under `specs/` (each with `spec.md` and `spec.json`)
- `.nojekyll` to prevent GitHub Pages from skipping `_sidebar.md`
- `.gitignore` for OS/editor artifacts
